1. Что такое Git?
    1) инструмент, позволяющий реализовать, распределённую систему контроля версиий             R
    2) сервис для хостинга IT проектов
    3) облачная PaaS платформа, поддерживающая ряд языков программирования
2. Что такое GitHub?
    1) веб-инструмент жизненного цикла DevOps с открытым исходным кодом
    2) система вопросов и ответов о программировании
    3) сервис для хостинга IT проектов          R
3, Что такое коммит?
    1) единица контента, хранящая в себе информацию о том, какие компоненты репозитория были изменены на текущий момент времени          R
    2) ветка или копия проекта, в которую можно вносить любые изменения и они не повлияют на основной проект
    3) отправка сделанной работы туда, где хранится код
4, Что такое репозиторий?
    1) место, где хранятся и поддерживаются какие-либо данные             R
    2) система управления версиями
    3) ветка или копия проекта, в которую нельзя вносить изменения
5. Git status - ...
    1) показывает содержимое файла
    2) показывает состояние репозитория
    3) показывает состояние проекта         R
6. Статус файла Untracked означает, что ...
    1) файл не отслеживается     R
    2) файл удалён
    3) файл изменён
7. Статус файла New означает, что ...
    1) файл вне репозитория
    2) файл изменён
    3) файл начал отслеживаться         R
8. Статус файла Modified означает, что ...
    1) такого статуса не существет в Git
    2) файл изменён         R
    3) файл добавлен в коммит
9. Как сделать коммит?
    1) git commit -n
    2) git add
    3) git commit -m        R
10, Что делает команда git log?
    1) показывает историю коммитов          R
    2) удаляет файлы из репозитория
    3) такой команды не существует
11. Как сделать клонирование проекта с GitHub на ПК?
    1) git new название проекта
    2) git add "ссылка на проект"
    3) git clone "ссылка на проект"         R
